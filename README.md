## How To Use

<!-- Example: -->

To clone and run this application, you'll need [Git](https://git-scm.com), [npm](http://npmjs.com) and [Yarn](https://yarnpkg.com/) installed on your computer. From your command line:

```bash
# Clone this repository
$ git clone https://github.com/jeevi/weather-app

# Install dependencies (you need have yarn installed beforehand)
$ yarn install

# Run the app (make sure port 3000 is free)
$ yarn start
```
